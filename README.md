# IronCore Fitness

**Live site:** https://frack777.github.io/ironcore-fitness/

## About

IronCore Fitness is a fictional gym website built for CST 2309 (Spring 2026). It uses HTML, CSS, and JavaScript to showcase a modern fitness center with three pages:

- **About Us** — mission, values, programs, and staff
- **Membership** — registration form with client-side validation ([presentation notes in `js/validation.js`](https://github.com/Frack777/ironcore-fitness/blob/main/js/validation.js) — search `PRESENTATION`)
- **Creativity** — interactive features like a photo slideshow, workout generator, event countdown, goal tracker, and light/dark mode
